프로그래스 바

            Progress bar : 작업의 진행 방향을 시각적으로 표현해주는 것
            진행하는 상황을 100%로 나타내서 사용자에게 작업이 얼마나 완료되었는지 보여줌

            단계가 있거나 대기시간이 필요한 작업에서 유용

            속성
            progress : 프로그래스바의 컨테이너 역할을 함
            progress-bar : 실제로 진행상황을 표시하는 부분
                ->부트스트랩에서 기본 색상과 애니메이션 제공

            aria-valuemin : 진행 시작 % 설정. 보통 0%
            aria-valuemax : 진행 끝 % 설정. 보통 100%
            aria-valuenow : 현재 진행률  설정.
                style : 프로그래스 바의 너비 설정을 여기서 주로 함
