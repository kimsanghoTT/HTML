localStorage

        인터넷에 데이터를 저장하는 방법 중 하나

        키 - 값의 쌍으로 데이터를 저장
        인터넷 세션간의 데이터를 유지
        인터넷을 닫았다가 열어도 저장된 데이터가 사라지지 않음

        localStorage.setItem(key, value) : 데이터를 저장
        localStorage.getItem(key) : 데이터를 조회
        localStorage.removeItem(key) : 데이터를 삭제
        localStorage.clear() : 모든 데이터를 삭제

        인터넷에서 방문기록 삭제, 캐시 삭제 같은 행동을 하면 모든 데이터가 사라짐
