block 형식

        - 화면에서 공간을 세로로 분할
        - block 형식의 태그
        1. h1 ~ h6 (제목)
        2. p/pre (문단) -> 글자 위주
        3. hr (수평선)
        4. div (영역분할) -> 글자, 텍스트, 그림 등 각종 파일 표현

        - 블록 형식 태그들은 작성된 내용이 짧더라도 한 줄을 모두 차지하려는 성질을 가짐
          때문에 다음 태그 내용은 다음 줄에 작성

div 형식

        - 가장 많이 사용하는 태그
        - 문서를 구분짓는데 많이 사용
        - div 안에 div를 쓰는 이중 div 많이 사용
        - div 옆에 작성하는 class는 head부터 foot까지 모두 사용가능
            p, h1, table 등 모든 태그에 사용 가능

        - class는 홈페이지를 꾸밀 때 주로 사용(style안에 사용)
첫 번째 div
두 번째 div
두 번째 div

inline 형식

        - 화면을 세로로 잘라서 나눔
        - 한 줄을 모두 차지하지 않음. 내부에 작성된 내용 크기만큼 차지
        - width, height와 같은 크기 지정 속성 사용 불가
        -line 형식 태그
        1. b (굵기)
        2. i (기울이기)
        3. mark (글 배경색)
        4. span (영역분할) 등

in line 형식은 가로를 세로로 분할하는 성질을 가지고 있어서 줄이 바뀌지 않고 계속 옆으로 작성된다
첫 번째 span 두 번째 span 세 번째 span

  • block 형식 : 가로를 모두 차지하는 영역
  • in line 형식 : 가로에서 내용을 작성해준 만큼만 영역을 차지
